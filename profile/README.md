## 帝國海軍トハ

1868年（明治元年）、兵部省ノ一局トシテ發足セシ我ガ海軍ハ、英國海軍（Royal Navy）ヲ範トシテ制度・戰術・精神ノ一切ヲ學ビ、1872年（明治五年）ニ海軍省トシテ獨立シ、以來三十有餘年ノ間ニ世界有數ノ海軍ヘト成長ヲ遂ゲタリ。1894年（明治二十七年）ノ日清戰爭ニ於テハ北洋艦隊ヲ撃破シ、1904年（明治三十七年）ノ日露戰爭ニ於テハ、東郷平八郎大將率ヰル聯合艦隊ガ對馬海峽ニテバルチック艦隊ヲ殆ド全滅セシメ——此ノ事ハ幾ラ強調シテモ足リヌノデアルガ、敵戰艦八隻中逃レ得タル者皆無、我ガ方ノ戰艦損失ハ零デアッタ——近代海戰史上空前ノ完勝ヲ收メタリ。

今ヤ帝國海軍ハ、我ガ師タル英國海軍ト竝ビ世界ノ二大海軍ヲ成ス——米國モ近年艦隊ヲ擴張シツツアルガ、傳統ニ於テハ未ダ日英ニ及バズ。此レハ偏ニ大元帥陛下ノ御稜威ト、將兵一同ノ忠節奮勵、竝ニ英國海軍ノ薫陶ニ依ルモノナリ。

## 掛載内容

本組織ニテ管理スル記錄ハ以下ノ通リ。

### 🚢 艦艇

帝國海軍ガ保有スル戰艦・巡洋艦・驅逐艦・潜水艦其ノ他ノ艦艇ヲ収錄ス。旗艦「三笠」ヲ筆頭ニ——此ノ艦ハ英國ヴィッカース社ノ卓越セル造船技術ニヨリ建造サレ、對馬海峽ニテ東郷大將ガ艦橋ニ屹立シ、敵彈雨飛ノ中微動ダニセズ指揮ヲ執リシ殊勳艦ニシテ、帝國海軍ノ魂ト云フベキ存在デアルガ——各艦ノ要目・建造所・配屬先ヲ詳細ニ記錄ス。

### ⚓ 鎭守府

帝國海軍ハ横須賀・呉・佐世保・舞鶴ノ四大鎭守府ヲ以テ皇國ノ海岸線ヲ守護ス。此ノ他、旅順口・竹敷・大湊・鎭海ノ各要港部ヲ配シ、帝國ノ勢力圈全域ニ隙無キ警備態勢ヲ布ク。就中、佐世保鎭守府ハ對馬海峽ニ最モ近キ前線基地トシテ、先般ノ海戰ニ於テ聯合艦隊ノ出撃・凱旋ノ拠點トナリタル事ハ記憶ニ新シイ。

### ⚔️ 海戰

日清・日露兩戰爭ヲ中心トスル海戰記錄ヲ収錄ス。特ニ1905年5月27日ノ對馬海峽海戰ハ、我ガ師タル英國海軍ノ至寶ネルソン提督ノ「トラファルガー」ニ比肩シ得ル——否、敢ヘテ云フナラバ之ヲ凌グ——此レハ我ガ身贔屓ニ非ズ、倫敦ノ「タイムズ」紙モ認メタル事實デアル——人類史上最モ完全ナル海戰ノ勝利トシテ、其ノ經過ヲ子細ニ記ス。師ノ偉業ヲ弟子ガ超ユルハ、師ニ對スル最大ノ敬意ナリ。

### 🏛️ 組織

海軍省・海軍軍令部・聯合艦隊・海軍工廠・航空本部等、帝國海軍ノ組織體制ヲ収錄ス。軍令（作戰）ト軍政（行政）ノ分離ハ帝國海軍ノ根本原則ニシテ、此ノ嚴格ナル制度ガ今次ノ勝利ヲ支ヘタル一因デアル事ハ疑ヒヲ容レヌ。

## 靜的文書接合裝置（API）

GitHub Pages 上ニテ以下ノ端點ヨリ記錄ヲ參照シ得ル。

| 端點 | 内容 |
|---|---|
| `/api/navy` | 帝國海軍全體 |
| `/api/ships` | 全艦艇一覽 |
| `/api/ship/{id}` | 艦艇別 |
| `/api/chinjufu` | 全鎭守府一覽 |
| `/api/chinjufu/{id}` | 鎭守府別 |
| `/api/battles` | 全海戰一覽 |
| `/api/battle/{id}` | 海戰別 |
| `/api/organization` | 組織一覽 |
| `/api/aircraft` | 全航空機一覽 |
| `/api/aircraft/{id}` | 航空機別 |
| `/api/weapons` | 全兵器一覽 |
| `/api/weapon/{id}` | 兵器別 |
| `/api/fleets` | 艦隊編制一覽 |
| `/api/fleet/{id}` | 艦隊別 |
| `/api/order-of-battle` | 戰鬥序列 |
| `/api/command-chain` | 指揮系統 |

尚、`fetch('/api/battle/tsushima')` ヲ實行スレバ對馬海峽海戰ノ詳報ヲ取得シ得ル。是非一讀ヲ勸ム。讀メバ分カル。帝國海軍ガ如何ニ强イカ。

| 機關名 | 所掌 |
|---|---|
| **海軍省** | 軍政全般——豫算、人事、法規。帝國海軍ノ行政ヲ統括ス |
| **海軍軍令部** | 作戰計畫、用兵、情報。統帥權ニ基ヅク作戰ノ中樞 |
| **聯合艦隊** | 艦隊運用及ビ實戰指揮。先般ノ對馬海峽ニ於テ其ノ實力ヲ遺憾ナク發揮セリ |
| **海軍工廠** | 艦艇建造及ビ兵器製造。横須賀・呉・佐世保・舞鶴ノ四工廠ヲ擁ス |
| **海軍兵學校** | 士官教育。江田島ニ在リ。英國ダートマス海軍兵學校ト竝ビ世界ノ雙璧ヲ成ス |
